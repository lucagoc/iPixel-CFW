# iPixel-CFW

Trying to make a custom firmware for LED Matrix controlled by iPixel Color App (Early State)

Identified Hardware:

- SPI ROM : ZBIT ZB25VQ32 ([Datasheet](https://www.lcsc.com/datasheet/C495743.pdf))

- MCU : JieLi AC6951C ([Datasheet](https://www.yunthinker.net/wp-content/uploads/2024/08/AC6951C-Datasheet-V1.7.pdf), [Discussion](https://github.com/lucagoc/iPixel-CLI/issues/13))
